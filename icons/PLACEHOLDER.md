# Icons

Place your icon files here:

- `icon16.png`  — 16×16 px  (toolbar / favicon)
- `icon48.png`  — 48×48 px  (extension management page)
- `icon128.png` — 128×128 px (Chrome Web Store)

Until real icons are added, you can generate temporary PNGs from any
SVG or use a tool like https://favicon.io. The extension will load
without icons but Chrome DevTools will log a warning.
